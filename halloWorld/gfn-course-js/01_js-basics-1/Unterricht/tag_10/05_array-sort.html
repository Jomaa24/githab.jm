Array Methode/Funktion - array.sort()

array.sort() - Die sort() Methode sortiert die Elemente eines Arrays in-place und gibt das Array zurück. Standardmäßig werden alle Elemente in Strings umgewandelt und dann anhand ihrer UTF-16 Codepoints miteinander verglichen, sprich lexikographisch sortiert.

Das angesprochene Array wird durch in-place-mutation ebenfalls sortiert.
