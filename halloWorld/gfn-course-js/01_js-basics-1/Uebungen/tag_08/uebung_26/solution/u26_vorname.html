Übung 26: Wenn ein Vorname nicht reicht …

Vielleicht ist dir aufgefallen, dass die Zerlegung nur richtig funktioniert, wenn der Name in der Form 'Vorname Nachname' vorliegt.

Sollte ein Nutzer einen zweiten Vornamen angeben, ordnet das Programm den zweiten Vorname dem Nachnamen zu. Du würdest für den Namen Ladislaus Coolio Jones fälschlicherweise Folgendes erhalten:

const firstName = 'Ladislaus'
const lastName = 'Coolio Jones'

Korrigiere die Ausgabe des folgenden Codes mithilfe von lastIndexOf
