Restparameter (spread operator ...)

spread-operator Die Spread-Syntax erlaubt es, einen einzelnen Array-Ausdruck oder String an Stellen zuexpandieren, an denen Null oder mehr Argumente (für Funktionsaufrufe) oder Elemente (für Array-Literale) erwartet werden, oder einen Objekt-Ausdruck an Stellen zu expandieren, an denen Null oder mehr Schlüssel-Wert-Paare (für Objektliterale) erwartet werden.
